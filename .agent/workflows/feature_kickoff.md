---
description: End-to-end workflow from idea to implementation
---

1. Define the Feature
// turbo
   - `/propose_features`
   - **Goal**: Create the `feature_proposal.md` artifact.

2. Create the Code
// turbo
   - `/scaffold_new_script`
   - **Goal**: Get a running file in `scripts/`.

3. Implement and Test
   - **Action**: Write your logic.
// turbo
   - `/development`
   - **Goal**: Ensure linting and tests pass.

4. Polish Documentation
// turbo
   - `/write_docstrings`
   - **Goal**: Add Google-style docstrings to the new code.

5. Update project docs
// turbo
   - `/update_docs`
   - **Goal**: Add the new script to guides or README.
