# Technology Stack

## Core
- **Language:** Python >= 3.10
- **Package Manager:** `uv`
- **Build System:** Setuptools

## Application Frameworks
- **GUI:** PySide6 (Qt for Python)
- **CLI:** Typer + Rich

## Google Integration
- **Google API Client:** `google-api-python-client`
- **Authentication:** `google-auth`, `google-auth-oauthlib`

## Development Tools
- **Testing:** Pytest
- **Testing Extensions:** `pytest-qt` (GUI), `vcrpy` (API Recording)
- **Mock Data Generation:** `polyfactory` (deterministic factories)
- **Interactive Debugging:** IPython
- **Linting & Formatting:** Ruff
- **Architectural Linting:** `import-linter`
- **Static Type Checking:** Mypy

## Data & Utilities
- **Data Analysis:** Pandas
- **Timezone Support:** `tzdata`
